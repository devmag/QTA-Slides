                                                                                                                    pablobarbera Mod Lynnie • 3 years ago

                                                                                                                    Hi Lynnie, thanks! Yes, I've found that query for more than 50 users at a time with 'getUsers' often gives errors. It appears the Facebook API doesn't like long queries. There are two ways around this:

                                                                                                                    1) If you only want the user data for your friends, you can run:

                                                                                                                    my_friends_info <- getFriends(token, simplify=FALSE)

                                                                                                                    That's equivalent to the two lines of code in my post, and will return all the private information for your friends.

                                                                                                                    2) A slightly more complicated way around this would be to split the process of getting the user data in chunks of 50 users:

                                                                                                                    my_friends <- getFriends(token)
                                                                                                                    my_friends_info <- list()
                                                                                                                    sf <- c(seq(1, length(my_friends$id), 50), length(my_friends$id))
                                                                                                                    for (i in 1:(length(sf)-1)){
                                                                                                                    cat(i, " ")
                                                                                                                    my_friends_info[[i]] <- getUsers(my_friends$id[sf[i]:sf[i+1]], token=my_oauth, private_info=TRUE)
                                                                                                                    }
                                                                                                                    my_friends_info <- do.call(rbind, my_friends_info)

                                                                                                                    I hope either of these two alternatives work for you! :)
